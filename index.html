例句：
AWS S3存储桶策略中，允许特定IAM用户读取桶内对象：
```json
{
"Version": "2012-10-17",
"Statement": [{
"Effect": "Allow",
"Principal": {"AWS": "arn:aws:iam::123456789012:user/alice"},
"Action": "s3:GetObject",
"Resource": "arn:aws:s3:::example-bucket/"
}]
}
```
